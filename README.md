# Jetson-Nano-temperature

Bash script for measuring temperature components.

### Download
```sh
$ git clone https://github.com/MartinMatta/Jetson-Nano-temperature
$ cd Jetson-Nano-temperature
$ chmod 755 temp.sh
```
### Run
```sh
$ ./temp.sh
AO-therm: 36.00°C
CPU-therm: 27.00°C
GPU-therm: 27.50°C
PLL-therm: 25.50°C
thermal-fan-est: 27.00°C
```
